beamon.core.assemble_bc(nodes_bc, Dof)[source]¶

Assemble boundary condition information in a vector. nodes_bc should contain [u_x,u_y,u_z,phi_x,phi_y,phi_z] for each node, which can only contain 0 (free) or 1 (locked). The assembled bc vector could be ex. [1,2,5,6] that locks movement in those numbered directions.

Parameters

  • nodes_bc (integer matrix) – boundary condition information from database

  • Dof (integer matrix) – nodes degrees of freedom

Returns

bc vector

Return type

integer 1 x n vector

beamon.core.make_joints(edof, nodes, dof, ndofs, bc)[source]¶

Extends edof, nodes, dof and bc matrices with new cloned nodes that has new dof. This will adjust translation conditions between elements. New nodes and dofs will be appended at the ende of each input ‘nodes’ and ‘dof’. edof matrix contain element dof numbers for each nodal point end. Free dof number is predefined as zero. nodes matrix contain [x,y,z] nodes coordinates in each row. dof matrix contain nodes degrees of freedom for each node in a row. bc array contain nodes degrees of freedom that are locked. map contains mapping information from cloned nodes to origin nodes

Parameters

  • edof (integer matrix) – topology matrix

  • nodes (float matrix) – nodes coordinates matrix

  • dof (integer matrix) – nodes dof matrix

  • ndofs (integer (3 or 6)) – number of dof in each node

  • bc (integer array) – boundary conditions array

  • map (dict) – origin and cloned nodes mapping

Returns

edof, nodes, dof, map

change_dof(index, u_x=None, u_y=None, u_z=None, phi_x=None, phi_y=None, phi_z=None)[source]¶
change the degree of freedom for a given node with specified node index. DOF specification:

1 is completely free, 0 is locked.

for example (index=300, u_x=1, u_y=1, u_z=1, phi_x=1, phi_y=1, phi_z=0) locks the rotation in the z axes of node 300.

Parameters

  • index – index of the node

  • u_x – movement in x direction

  • u_y – movement in y direction

  • u_z – movement in z direction

  • phi_x – rotation in x axes

  • phi_y – rotation in y axes

  • phi_z – rotation in z axes

Returns

True: if successfully changed/ False: if node not found
